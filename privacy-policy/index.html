1. Introduction

PingMe is a real-time peer-to-peer messaging application. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our application.

Please read this Privacy Policy carefully. If you do not agree with our policies and practices, please do not use our App. By using PingMe, you accept and agree to be bound by this Privacy Policy.

2. Information We Collect

2.1 Information You Provide Directly

  • Account Information: Username, display name, profile picture
  • Message Content: All messages and shared files (stored locally)
  • Contact Information: Device name and network identifier
  • Authentication Data: If using optional cloud features (email, password)

2.2 Information Collected Automatically

  • Device Information: Device type, model, OS version, unique device ID
  • Network Information: WiFi network name (SSID), IP address
  • Location Data: Approximate location from WiFi network (for device discovery)
  • Usage Data: Features used, chat frequency, app performance metrics
  • Camera/Media: Only with explicit user permission and consent

2.3 Information NOT Collected

  • Call recordings or transcriptions
  • Real-time location tracking
  • Personal browsing history
  • Bank or financial information
  • Biometric data (unless user explicitly enables)

3. How We Use Your Information

3.1 Core App Functions

  • Enable peer-to-peer messaging with users on same network
  • Discover and display available nearby devices
  • Deliver messages in real-time
  • Store messages locally for offline access
  • Send push notifications for new messages

3.2 App Improvement

  • Analyze app crashes and performance issues
  • Understand user behavior patterns
  • Improve user experience and features
  • Test new features with user consent

3.3 Communication

  • Send app updates and announcements
  • Respond to user support requests
  • Notify users of policy changes

3.4 Security

  • Prevent fraud and unauthorized access
  • Enforce our Terms of Service
  • Comply with legal obligations

4. Data Storage & Security

4.1 Local Storage

Messages are encrypted and stored locally on your device. Messages are NOT transmitted to external servers. Clearing app data will delete all local messages. Users can export their message history.

4.2 Cloud Storage (Optional)

If user enables cloud backup, data is encrypted end-to-end. Cloud data stored on Firebase Firestore (Google). User can disable cloud sync at any time. Google's privacy policy applies to Firebase data.

4.3 Encryption

  • Messages encrypted using industry-standard encryption (AES-256)
  • User credentials protected with hashing
  • HTTPS/TLS for all network communications
  • Public key infrastructure for peer verification

4.4 Data Security Measures

  • Regular security audits and penetration testing
  • Encrypted communications between devices
  • No plaintext storage of sensitive data
  • Automatic session timeouts
  • Permission-based access control

5. Data Sharing & Third Parties

5.1 We Do NOT Share

  • Personal information with advertisers or marketers
  • User messages with any third party
  • Location data beyond what's needed for network discovery
  • Data for profiling or targeted advertising

5.2 We DO Share Data With

  • Firebase (Google): If cloud features are enabled
  • Crash Reporting Service: Error logs (anonymized)
  • Law Enforcement: Only with valid legal warrant or court order
  • Other Users: Only messages you explicitly send to them

6. User Rights & Choices

6.1 Data Access

  • Users can view all their stored messages anytime
  • Export full message history in standard format
  • Request copy of personal data we hold

6.2 Data Deletion

  • Delete individual messages or conversations
  • Clear all app data via app settings
  • Delete account and all associated data (cloud)
  • Data deleted from local device cannot be recovered

6.3 Permissions Control

  • Users can grant/revoke permissions anytime in settings
  • App gracefully handles permission denials
  • No required permissions for basic chat functionality
  • Optional permissions clearly marked

7. Children's Privacy

This App is NOT directed to children under 13 years of age:

  • We do not knowingly collect data from children under 13
  • If we learn we've collected such data, we'll delete it immediately
  • Parents/guardians: Contact us if concerned about underage usage
  • Parental controls: Device-level parental controls should be used
